Excel Visual Basic for Applications

I developed a tool that was used during the review of video data using Visual Basic for Applications(VBA) on Microsoft Excel. The objective was to make it easier for the user to apply the methodology while reviewing videos. The tool has many useful features for the user such as:

  • Opening specified videos for review
  • Replaying events
  • Jumping seconds backwards and forward
  • Rate of playing speed
  • Deleting rows in the excel
  • Recording times
  • Filling the methodology in excel with just a click of a button
